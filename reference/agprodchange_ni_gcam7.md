# agprodchange_ni_gcam7

Agricultural productivity change (unitless) obtained from GCAM7
Reference scenario. This is the baseline APC that is used to calculate
the climate impacted APC.

## Usage

``` r
agprodchange_ni_gcam7
```

## Format

R data frame

## Source

https://jgcri.github.io/gcam-doc/inputs_supply.html

## Examples

``` r
if (FALSE) { # \dontrun{
 library(gaia);
 agprodchange_ni_gcam7 <- gaia::agprodchange_ni_gcam7
} # }
```
