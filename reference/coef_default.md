# coef_default

Regression coefficients generated by
[\`broom::tidy()\`](https://broom.tidymodels.org/) for the regression
model using gaia's example historical climate data, including estimate,
standard error, statistic, p-value, for the 17 default crops supported
by gaia.

## Usage

``` r
coef_default
```

## Format

R data frame

## Source

Regression coefficients derived by using the default historical WATCH
climate data

## Examples

``` r
if (FALSE) { # \dontrun{
 library(gaia);
 coef_default <- gaia::coef_default
} # }
```
