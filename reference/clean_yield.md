# clean_yield

Function to clean FAO yield data

## Usage

``` r
clean_yield(fao_yield = NULL, fao_to_mirca = NULL)
```

## Arguments

- fao_yield:

  Default = NULL. Data frame for the fao yield table

- fao_to_mirca:

  Default = NULL. Data frame for the fao to mirca crop mapping

## Value

A data frame of formatted FAO yield data
