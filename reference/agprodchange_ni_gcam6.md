# agprodchange_ni_gcam6

Agricultural productivity change (unitless) obtained from GCAM6
Reference scenario. This is the baseline APC that is used to calculate
the climate impacted APC.

## Usage

``` r
agprodchange_ni_gcam6
```

## Format

R data frame

## Source

https://jgcri.github.io/gcam-doc/inputs_supply.html

## Examples

``` r
if (FALSE) { # \dontrun{
 library(gaia);
 agprodchange_ni_gcam6 <- gaia::agprodchange_ni_gcam6
} # }
```
